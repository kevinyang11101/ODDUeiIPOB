# 在线投稿系统+SSM

## 前言

欢迎来到我们的在线投稿系统项目，本项目采用了SSM框架，即Spring、SpringMVC、MyBatis，结合微信小程序、前端Uniapp等技术，旨在为用户提供一个便捷、高效的在线投稿平台。

## 内容介绍

在线投稿系统为广大作者提供了一个便捷的投稿途径，可以实时跟踪投稿进度，提高作者与编辑的沟通效率。系统主要包括：用户注册登录、投稿管理、稿件审核、通知公告等模块。通过本系统，作者可以轻松投稿，编辑可以高效处理稿件，实现双方共赢。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis，微信小程序
- 前端技术：JS、Vue、CSS3，Uniapp
- 开发工具：IDEA/Eclipse，Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
